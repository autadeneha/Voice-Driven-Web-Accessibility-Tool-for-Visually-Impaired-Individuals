# Voice-Driven Web Accessibility Tool for Visually Impaired Individuals

## Overview

Voice-Driven Web Accessibility Tool is an AI-powered Chrome Extension designed to improve digital accessibility for visually impaired users. The system enables hands-free website interaction through voice commands and provides spoken feedback using text-to-speech technology.

The extension extracts webpage content, interprets images through alternative text descriptions, generates AI-powered summaries, and assists users in navigating websites efficiently without relying on traditional keyboard-based interactions.

---

## Problem Statement

Visually impaired individuals often face challenges while accessing web content due to limitations in traditional screen readers, complex navigation structures, and insufficient descriptions of visual elements.

This project aims to provide an intelligent voice-based accessibility assistant that simplifies web browsing and promotes digital inclusivity.

↓
Text-to-Speech Engine
↓
Audio Response

---

## Project Workflow

1. User provides a voice command.
2. Speech Recognition converts speech into text.
3. Intent Engine identifies the user's request.
4. Relevant webpage content is extracted using DOM processing.
5. OpenAI processes content for summarization or understanding.
6. Speech Synthesis converts output into audio.
7. User receives spoken feedback.
